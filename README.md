# FinTrack CLI (Python)

FinTrack is a simple and fast **command-line expense tracker** built using Python.  
It stores expenses in a JSON file and allows users to add, view, and analyze their spending.

This project is perfect for:
- Python beginners  
- Internship portfolio  
- Demonstrating file handling, CLI design, and logic building
